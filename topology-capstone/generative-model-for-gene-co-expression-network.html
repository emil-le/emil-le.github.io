Chapter 5 Generative model for gene co-expression network

5.1 Gene Expression

The task of classifying the cancer regulatory pathways of patients becomes important, and mathematics have provided us the tools to do so. The gene expression is one of many valuable biometric information to identify cancer cells, in particular, the mRNA copies presented in tumors. From this data, one can construct the gene co-expression network in order to analyze the behavior of active cancer genes. This capstone focuses on analyzing the difference of topology throughout AML networks in order to see if one can predict or dentify the biomark of AML to improve the prognosis of the disease or discover effective drug pathways.

Define the gene co-expression network that we want as \(G = (\mathcal{V}, \mathcal{E})\), where the vertex set \(\mathcal{V}\) consists of genes. An edge in the edge set \(\mathcal{E}\) denotes if there is similarity in expression level between 2 genes, which results in G being an undirected network.

5.2 Data Collection and Pre-processing

Computations is done mainly through R with visualization done through Gephi. I used the gene expression dataset from The Cancer Genome Atlas (TCGA) for both the training set (TCGA-LAML) and the validation set (query sample from healthy bone marrow and blood tissues). The data is obtained experimentally through Illumina HiSeq 2000 RNA Sequencing, which are \(log_2(x+1)\)-RSME normalized with 95% credibility interval.
