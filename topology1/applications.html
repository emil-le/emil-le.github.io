3.1 Modeling noisy image segmentation

This section is to demonstrate an application from discrete Morse theory for image segmentation. The segmentation of medical image (such as CT scan or MRI) requires high accuracy, however, image noises make this task very difficult as clusters could have broken edges or inaccurate gaps. We we implement the segmentation model that are robust to noise using methods from (). The architecture of the algorithm can be simplified as: \[\text{Input Image} \longrightarrow \text{Triangulation } \longrightarrow \text{Density Map } \rho \longrightarrow \text{Persistence diagram }\]

From the input image, we can represent each pixel as a point in space. For this model, we assume a triangulation \(\mathcal{K}\) from the input images and we only consider the \(0-\)simplex, \(1-\)simplex, and \(2-\)simplex. We extract the manifold from \(\mathcal{K}\) by the grayscale density map \(\rho: \mathcal{K}\to \mathbb{R}\), in this case, we choose the Laplacian of Gaussian spatial filter for the density map. Lastly, we apply the sublevel set filtration to compute the persistence diagram for the critical points of the diagram for noise modeling, critical points that have long survival interval represent the stable feature of the image, while those have short survival interval accounts for noise.

We used brain tumour MRI and histopathologic colorectal dataset from Kaggle to demonstrate the algorithm.

3.2 Gene co-expression network analysis

In this section, we try to identify the topological structure of the gene co-expression networks belong to patients with bipolar disorder or schizophrenia disorder. Bipolar disorder and schizophrenia are psychiatric disorders that has high mortality rates, which makes the task of diagnosis crucial to improve life quality. Symptoms of bipolar disorder includes manic episodes, which are characterized by increased energy with intense delusion thoughts, while schizophrenia disorder includes psychosis, which are characterized by hallucinations and detachment to reality. In general, different symptoms of psychiatric disorders can sometime look similar enough despite being completely separated, which leads to misdiagnosis. Thus, we want define the problem statement, whether using applications from discrete Morse theory and TDA, we can identify significant biomarkers of these two disorders.

\[ \begin{aligned}\text{Figure 4.1:} &\text{ Example of a gene co-expression network.} \end{aligned} \]
We collect the gene co-expression network data from NCBI. The gene co-expression network is represented as an undirected network \(G = (V, E)\), where the vertex set \(V\) consists of genes and an edge in the edge set \(E\) represents the similarity in expression level between 2 genes. A clique of a network \(G = (V, E)\) is a sub-network \(C\subset G\), where two distincts vertices in \(C\) are connected by an edge. We used the methods introduced by Kannan et. al.Â to capture the persistence cliques presented in the gene co-expression networks to compare the two psychiatric disorders. Let \(g_\max\) denotes the maximum degree of \(G\), we define the discrete Morse function on the vertex set of\(G\), \(f: V(G)\to\mathbb{R}\), as: \[f(v) = g_\max - \deg(v) + \epsilon\] where \(\epsilon\) are randomly generated from the uniform distribution \(\text{Uniform}(0,0.5)\).
